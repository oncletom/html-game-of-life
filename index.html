Game of Life
Playing with HTML and DOM.

This is a real life example of Conway's Game of Life, built during the 2011's Global Day of Code Retreat in Bordeaux.

I bet using HTML, JavaScript and DOM was one of the easiest way to achieve the goal, both programmatically and visually.
